Uganda PPT editable-hybrid pipeline

핵심 원칙: Image 2.0 계열 생성물은 완성 슬라이드가 아니라 visual insert 삽화로만 만들고, 제목·본문·질문·표·카드·교사노트는 수정 가능한 PowerPoint 도형/텍스트로 생성한다.

Status: ready-go · lessons 1,800 · packs 30 · editable slides 14,400 · image prompts 16,200 · mode editable_hybrid

Pipeline order

  1. Source ingest: P6/P7 txt 교재, NotebookLM-Slide Prompt.html, Uganda design guide를 읽고 학년·학기·과목·차시 메타데이터를 정리한다.
  2. Lesson catalog: canonical lesson 1,800개를 만들고 중복 suffix 파일 158개를 제외한다.
  3. Pack split: grade × term × subject 기준으로 30개 pack을 만든다. 1 pack은 보통 60 lessons, 480 slides, 540 image assets다.
  4. Prompt queue: 각 슬라이드와 별도 infographic에 대해 visual_insert 이미지 프롬프트를 생성한다.
  5. Image generation: 이미지 모델은 삽화만 만든다. 이미지 안에는 제목, 숫자, 라벨, 표, 카드, footer, fake text를 넣지 않는다.
  6. Editable PPT render: generated image를 PPT picture object로 삽입하고, 수업 구조는 python-pptx 코드가 editable shapes/text로 그린다.
  7. PDF export: 확인용 PDF를 만들어 줄맞춤, 칸 배치, 화면 품질을 검수한다.
  8. QA gate: 슬라이드 수, 이미지 panel 위치, editable text layer, 과목별 frame, teacher note, PDF 정렬을 확인한다.
  9. Pack ZIP: PPTX, PDF, images, manifest, QA report를 pack별로 묶는다.
  10. Publish: GitHub Pages와 release asset에 pack 단위로 게시한다.
